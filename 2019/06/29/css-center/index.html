竖直居中

flex

标准的竖直居中方式，适用于 IE10+ 和 现代浏览器

line-height

  • 必须是单行

  • 子元素必须是行内元素(inline, inline-block)

  • 父元素已知高度

::before ::after

  • 子元素必须是行内元素

calc

  • 已知子元素的高度

  • 竖直居中的 top 值为 50% - (height / 2)

transform

通用的竖直居中，适用于 IE9+ 和 现代浏览器

absolute

  • 因为绝对定位的元素是会互相覆盖的，所以只适用于单个子元素

  • 必须指定子元素的高度，子元素没有高度就是 100%

table

模拟 table 的竖直居中
