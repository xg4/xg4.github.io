commit message

  • feat: 新功能

  • fix: bug 修复

  • docs: 文档修改

  • style: 不影响代码的更改（如空格，格式，缺少分号等）

  • refactor: 代码重构，既不修复错误也不添加功能

  • perf: 代码更改可以提高性能

  • test: 添加缺失或更正现有测试

  • build: build 相关，npm scripts

  • ci: CI 相关

  • chore: 对构建过程或辅助工具和库（如文档生成）的更改

  • revert: 版本回退
